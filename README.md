# Digital Library Manager

A multi-user Flask web application for managing personal book collections.

## Features

- User registration and login
- Session authentication
- Add books
- Edit books
- Delete books
- Favorite books
- Search books
- Filter by category
- Reading statistics
- Dark mode UI

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS

## Installation

```bash
pip install flask
```

## Run Project

```bash
python app.py
```

Then open:

```plaintext
http://127.0.0.1:5000
```
